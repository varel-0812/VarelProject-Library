# 📚 Project Library - PHP Portfolio Website

Website portfolio untuk menampilkan project-project PHP dengan desain elegan, sistem scrolling, dan fitur CRUD lengkap.

## ✨ Fitur Utama

### Frontend:
- Desain modern dengan animasi smooth
- Particle.js background yang interaktif
- Sistem filtering projects berdasarkan kategori
- Counter animation yang dinamis
- Parallax effect saat scrolling
- Halaman detail project
- Fully responsive (mobile friendly)

### Admin Panel:
- Login system sederhana
- Dashboard dengan statistik real-time
- CRUD operations lengkap (Create, Read, Update, Delete)
- Form validation
- Search functionality
- Preview gambar real-time
- Quick actions
- Manajemen featured projects
- Export data ke CSV

## 🛠️ Teknologi yang Digunakan

- **PHP** (Native OOP)
- **MySQL** Database
- **HTML5** & **CSS3**
- **JavaScript** (Vanilla)
- **AJAX** untuk interaktivitas
- **Particles.js** untuk background efek
- **AOS** (Animate On Scroll)

## 📋 Requirements

- PHP 7.4 atau lebih baru
- MySQL 5.7 atau lebih baru
- Web Server (XAMPP/Laragon/WAMP)
